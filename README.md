# Butcher's Board

Website for Butcher's Board, built on Craft CMS 5 with Vite, Tailwind 4, and Swup page transitions.

## Tech stack

- Craft CMS 5 (Twig templates)
- Vite + `vite-plugin-craftcms`
- Tailwind CSS v4 (CSS-native `@theme {}` tokens)
- Swup (SPA-style page transitions)
- Imager X (image transforms)
- DDEV (local environment)

## Requirements

- [Docker](https://www.docker.com/)
- [DDEV >= 1.22.0](https://ddev.com/)

## First-time setup

```bash
ddev start
ddev composer install
ddev npm install
ddev craft install
ddev npm run start
ddev launch
```

`app/.env` is generated by `composer install`; double-check `PRIMARY_SITE_URL` and update any Servd values you need.

## Daily workflow

```bash
ddev start
ddev npm run start
ddev launch
```

## Environment configuration

- Copy `app/.env.example` to `app/.env` if Composer has not already done so.
- Set `PRIMARY_SITE_URL` to your local DDEV URL (`https://butchers-board.ddev.site`).
- Servd values (`SERVD_PROJECT_SLUG`, keys, SMTP) are optional — fill them in when you deploy to Servd.

## Common commands

- Clear Craft caches: `ddev craft clear-caches/all`
- Apply project config: `ddev craft project-config/apply`
- Update plugins and Craft: `ddev composer update`
- Lint JS: `ddev npm run lint`
